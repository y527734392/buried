# 介绍

埋点统计由两部分组成。
一个基础信息统计，包括但不限于页面请求时间、白屏时间、渲染时间、dom渲染时间等；一个业务相关的用户操作、接口数据的统计。
由埋点统计生成的数据，可以通过对应的数据观察出最近系统的变化及用户常规操作习惯，提供出一套出色的可视化的数据，方便各端人员观看。

# 背景

财务部门，目前不具备数据可视化，前端无法获得基本的数据信息。

# 运行原理

埋点统计实际是由 vue vueRouter webpack驱动的单页里面迁入的一套JavaScript代码。如果你JavaScript，那么可以进行更高级别的扩展及优化。

# 特性

  • 简单、开箱即用
  • 可定制、扩展更丰富的功能
