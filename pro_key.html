# 业务通用字段

# 介绍

业务字段，每个业务都会发送一些请求，所以定义通用字段

# 说明

# atnd

接口api 请求时间戳(日期)

    "atnd" : 15021321321321

# at

接口api 请求耗时 毫秒数

    "at" : 3232

# atst

接口api 请求返回状态（成功，失败）

    "atst" : "success"
    "atst" : "fail"

# au

接口api地址

    "au" : '/api/sms'
