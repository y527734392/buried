# 公共字段

# 介绍

公共字段定义是每次发送请求必带的一些数据，包含但不限pid、ref、ua、os、v；这些基础字段会统一维护在基础配置中，如非必要尽量不要修改。

# 基础

# pid 800-900

预留800； 以备后续其他系统接入

801 财务审批系统 （旧版-8001 后期废弃）
802 结算系统 （旧版-8002 后期废弃）
803 服务商结算系统 （旧版-8003 后期废弃）
804 资金管理系统
805 收银台系统
806 账户运营系统
807 支付基础配置系统
808 协议代扣系统
809 合同中心系统

# ref

来源 系统名称

rrcp-fiance         财务审批系统 （旧版 rrc-cw-fiance-old 后期废弃）
rrcp-settle         结算系统 （旧版 rrc-cw-settle-old 后期废弃）
rrcp-financing-mng  服务商结算系统 （旧版 rrcp-financing-mng-old  后期废弃）
rrcp-financing      资金管理系统
rrcp-paycenter      收银台系统
rrcp-acm            账户运营系统
rrcp-config         支付基础配置系统
rrcp-dk             协议代扣系统
rrcp-contract       合同中心系统

系统信息

Mozilla/5.0 (iPhone; CPU iPhone OS 11_3 like Mac OS X) AppleWebKit/605.1.15 (KHTML, like Gecko) Version/11.0 Mobile/15E148 Safari/604.1

    "os": "Andiord", // 安卓
    "os": "IE", // ie 浏览器
    "os": "Chrome", // 谷歌 浏览器
    "os": "firefox", // 火狐 浏览器
    "os": "QQ", // qq 浏览器
    "os": "2345", // 2345 浏览器
    "os": "Sougo", // 搜狗 浏览器
    "os": "other", // 其他 浏览器

# v

当前业务版本号，获取为package中的version

    "v" : "v1.0.0"

# 非基础必要

# uid

获取登录用户id，记录用户唯一性，后期过滤，即可获得该用户的一些行为习惯

改字段，需要加密， charCode 平移1个单位，获取数据 偏移回来即可。

    "uid": "12678"

# uname

获取登录用户真是姓名

改字段，需要加密， charCode 平移1个单位，获取数据 偏移回来即可。

    "uname": "张三"

# snd

埋点数据上报日期 （yyyy-MM-dd HH:mm:ss）

    "atnd" : '2018-09-01 18:00:00'

# type

统计的类型

    "type": "FEPageTime", // 前端 页面加载时间

# page

页面名称

    "page": "list"  // 前端页面 路由name

# page_url

    "page_url": "http://rrc.com/list" // 前端页面 详细链接
